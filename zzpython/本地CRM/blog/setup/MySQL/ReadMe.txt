数据库使用说明
==============
博易的运行是不需要专门的数据库的，它的数据默认全部存放在一组XML里面。为了满足用户更多的需求，博易又支持了扩展数据库。这就意味着您可以将多种数据库用来支持博易的运行。下面我们给出如何将博易运行于 MySQL 5 之上的解决方案。 

1.创建一个 MySQL 数据库，运行这个 SQL 脚本（MySQLSetup1.4.5.0.sql）。 

2.备份根目录下的 Web.config 文件。

3.重命名本目录下的 MySQLWeb.Config 为 Web.config 并覆盖程序目录下的 Web.config 文件（此操作前请先备份原文件）。

4.运行并更改密码（初始账号 admin 密码 admin）。

只需完成以上4个步骤即可！ 


将博易v1.6的数据库升级到博易v1.6.5的解决方案
============================================
打开博易运行的数据库，运行这个 SQL 脚本（MySQLUpgradeTo1.4.5.0From1.4.0.0.sql）即可。
