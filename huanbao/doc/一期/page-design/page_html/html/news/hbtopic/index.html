广西龙江发生镉污染事件
    2012年1月15日，宜州市环保部门检测发现龙江河宜州拉浪段重金属超标，到19日镉浓度仍超标3倍，26日污染水体进入下游柳江系统，27日柳江上游河段镉浓度超标达5倍。广西龙江河突发环境事件应急指挥部30日召开新闻通气会称，龙江河镉污染应急处置取得成效，事态完全处于控制之中，下游柳江水质仍达标。污染事件使沿江居民生活受到严重影响，下游柳州民众已大规模抢购瓶装水。目前，广西镉污染源仍未确定...[详细]
